 Grounding Grid: Analysis grid discretization

Victor Batista January 2, 2025

The solution in grounding grid analysis depends directly on the grid discretization. The smaller the segment, the more faithful the representation of reality.

In this study, I simulated a 100x100m grid with 10x10m grids with different segment sizes. The graph shows that reducing the segment size leads to a more uniform voltage distribution that is closer to the real value.
